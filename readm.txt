# 📚 Détection de Malware avec Attaques Adversariales

## 🚀 **Description du Projet**
Ce projet vise à détecter les logiciels malveillants avec un modèle robuste face aux attaques adversariales (**FGSM, PGD, DeepFool**) et amélioré grâce à **SMOTE**.

## 🛠️ **Technologies Utilisées**
- Python (Scikit-learn, TensorFlow, Keras)
- SMOTE pour équilibrer les classes
- Streamlit pour le déploiement
